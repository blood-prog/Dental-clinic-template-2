# Ayine Care

A modern medical clinic website and admin dashboard with a serene, editorial aesthetic. Built with Tailwind CSS, Libre Caslon Text + Manrope typography, and the Serene Medical Collective design system.

## Features

- Patient-facing landing page with services, FAQ, and online booking
- Password-protected admin dashboard with appointments CRUD, analytics, and CSV export
- Visit tracking and conversion analytics
- Upstash Redis backend via Vercel serverless functions

## Deploy

1. Connect this repo to Vercel
2. Add Upstash Redis from the Vercel Marketplace
3. Set `DASHBOARD_PASSWORD` environment variable
4. Deploy
